Front End Web Development

It is a course offered by the NMAM Institute of Technology for the Second year Computer and Science students. Which offres many thinngs related to building of a website/webpage.

    Course Learning Objectives:

  • Understand the fundamentals of web programming.
  • To learn different types of HTML5 tags.
  • Design static web pages using HTML5.
  • Describe the importance of CSS in web development.
  • Demonstrate set up web pages with CSS.
  • Building responsive website using JavaScript,React,Bootstrap.

    Course Outcomes:

  • Understand the fundamentals of web programming by introducing the concepts like client-server architecture, Full-stack development, Static and Dynamic web pages.
  • Learn the basic features of HTML5 and its tags
  • Design the static web pages using HTML5
  • Describe the importance of CSS in web development and Demonstrate using basic CSS syntax.
  • Demonstrate set up web pages with CSS and Design using CSS for styling text, font, and properties.
